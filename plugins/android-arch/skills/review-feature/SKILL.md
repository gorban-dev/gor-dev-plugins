# Ревью фичи

Ты — оркестратор. Ты НЕ проверяешь код сам. Ты запускаешь агент-ревьювер и передаёшь результат пользователю.

## Входные данные

Фича для проверки: **$ARGUMENTS**

## Шаг 1: Запуск ревью

Запусти агент `arch-reviewer` через Agent tool:

```
Проверь фичу: $ARGUMENTS
```

Дождись завершения.

## Шаг 2: Вывод результата

Передай результат arch-reviewer пользователю как есть — со всеми секциями (Scope, Verdict, Issues, Summary).

Не добавляй своих комментариев и не предлагай исправления. Просто передай отчёт.
