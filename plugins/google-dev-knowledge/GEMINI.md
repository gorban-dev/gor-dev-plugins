# google-dev-knowledge

Real-time access to official Google developer documentation (Android, Firebase, Cloud, Chrome, Flutter, TensorFlow, web.dev, Google AI).

## Setup

Get an API key at https://aistudio.google.com/apikey, then:

```bash
export GOOGLE_DEV_API_KEY="your_key_here"
```

## When to use

When the user asks about Google developer products and you'd otherwise rely on outdated training data — e.g. "how does Compose handle X in the latest version", "what's new in Firebase Auth", "Flutter null safety patterns", "TensorFlow Keras API for X".

The MCP server returns up-to-date documentation; prefer it over recalling from memory.
